# cheatshees

## general

```
ctrl + t: Open a new tab in the browser
ctrl + w: Close the current tab in the browser
ctrl + f: Find text in the current document or webpage
ctrl + c: Copy selected text or items
ctrl + v: Paste copied text or items
ctrl + x: Cut selected text or items
ctrl + b: Bold selected text
ctrl + i: Italicize selected text
ctrl + u: Underline selected text
ctrl + z: Undo the last action
ctrl + shift + z: Redo the last undone action
ctrl + a: Select all text or items in the current document or webpage
ctrl + s: Save the current document or webpage
ctrl + space: switch input method
ctrl + r: Refresh the current webpage or document
ctrl + l: Focus on the address bar in the browser
```

## Windows

```
win + down: Minimize the current window
ctrl + shift + f: switch input method from simplified to traditional Chinese
win + shift + s: Take a screenshot and copy it to the clipboard
win + left: Snap the current window to the left half of the screen
alt + tab: Switch between open applications
```

## mac

```
command + k: clip photo
command + option + d: Show or hide the Dock
```

## ios

```
search key in the home screen: search apps, contacts, settings, etc.
```

## excel

```
F2: Edit the active cell
ctrl + shift + "+": Insert a new row above the current row
ctrl + space: Select the entire column
ctrl + shif: Select the entire row
alt + H + O + I: Auto-fit the width of the selected column
alt + H + O + A: Auto-fit the height of the selected row
shift + mouse drag: put the selected column to other place
```

## teams

```
ctrl + e: Focus on the search bar
ctrl + 2: Go to the Chat tab
ctrl + f: Search for a message or file
esc: Close the current search or menu
```

## mobaxterm

```
ctrl + alt + t: Open a new tab
ctrl + alt + q: Close the current tab
ctrl + shift + f: Find text in the current terminal
shift + mouse drag: Select text in the terminal
ctrl + scroll wheel: Zoom in or out the terminal font size
```

## my vim

```
s + hjlk: split window
tu,th,tl: open and switch between tabs
```

## tmux

```
prefix + c: Create a new window
prefix + number: Switch to the specified window
prefix + [: Enter copy mode
prefix + ]: Paste the copied text
prefix + d: Detach from the current session
```
