# tar(1)


```
tar -cf <archive> <files> # create archive
tar -cvzf <archive> <files> # create archive with gzip compression. (tar.gz, tgz)
tar -cvjf <archive> <files> # create archive with bzip2 compression. (tar.bz2, tbz)
tar -xvf <archive> # extract archive
tar -xvzf <archive> # extract archive with gzip compression
tar -xvjf <archive> # extract archive with bzip2 compression
tar -tf <archive> # list archive contents
```
